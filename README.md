## 🚀 Overview
**Quiz App** is a simple browser‑based quiz game that presents a set of questions to the user, evaluates answers, and shows the final score. This project focuses on **DOM manipulation, event handling, and JavaScript logic** — making it a great example of real‑world interactive frontend behavior without frameworks.

---

## 🧠 Features
- 🎯 Multiple choice questions
- 📝 Instant feedback for each answer
- 🧮 Final score display
- 🔄 Reset quiz & try again
- 📱 Responsive layout for various screen sizes
